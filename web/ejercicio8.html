Ejercicio 8: Joiners

Descripción

Este ejercicio muestra cómo se implementan las interfaces genéricas en Java y cómo se pueden utilizar para manipular colecciones de elementos. En este caso, se tiene una interfaz Joiner que permite combinar elementos de tipo String o Integer de diferentes formas.

Código de la clase Ejercicio8.java


package org.iesvdm.transformer;

import java.util.ArrayList;
import java.util.Scanner;

public class Ejercicio8 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        ArrayList a = new ArrayList<>();
        for (int i = 0; i < 10; i++) {
            a.add(i);
        }

        ArrayList b = new ArrayList<>();
        b.add("A");
        b.add("B");
        b.add("C");
        b.add("D");
        b.add("E");
        b.add("F");
        b.add("G");
        b.add("H");
        b.add("I");
        b.add("J");

        System.out.println(a);
        System.out.println(b);
        Joiners j = new Joiners();
        JoinByAdding j2 = new JoinByAdding();
        j.fold(j2, a);
        JoinSpace j3 = new JoinSpace();
        j.fold(j3, b);

        sc.close();
    }
}

Explicación

El código en Ejercicio8.java crea dos listas: una de enteros y otra de cadenas. A continuación, utiliza la clase Joiners y sus métodos para unir los elementos de estas listas de distintas formas:

  • Usando la clase JoinByAdding para sumar los valores de la lista de enteros.
  • Usando la clase JoinSpace para concatenar los elementos de la lista de cadenas, separándolos con un espacio.

Código de la clase JoinByAdding.java


package org.iesvdm.transformer;

class JoinByAdding implements Joiner<Integer> {
    public Integer join(Integer int1, Integer int2) {
        return int1 + int2;
    }
}

/*
 * <------------------------- Explicacion JoinByAdding -------------------------> 
 * En el archivo JoinByAdding, hay una clase que implementa la interfaz Joiner<Integer>,
 * junto a un metodo join, que recibe dos enteros, y devuelve la suma de ambos.
 */

Explicación de JoinByAdding

En la clase JoinByAdding, la interfaz Joiner<Integer> es implementada para realizar la suma de dos números enteros. El método join toma dos parámetros de tipo Integer y retorna su suma.
