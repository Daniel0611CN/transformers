Ejercicio 1 - Multiplicación de Elementos

En este ejercicio, se multiplica cada elemento de un ArrayList de números por un factor constante. La clase Transformer es usada para implementar la transformación en cada elemento.

Código Principal (Ejercicio1.java)

public class Ejercicio1 {
    public static void main(String[] args) {
        ArrayList<Integer> array = new ArrayList<>();
        // Código para introducir y transformar valores
        Transformer<Integer> transformers = new Transformer<>() {
            @Override
            public Integer transform(Integer obj) {
                return obj * 2;
            }
        };
        ArrayList<Integer> arrayTransformado = Transformers.applyConst(transformers, array);
        System.out.println(arrayTransformado);
    }
}

Clases Relacionadas

Transformers.java: contiene los métodos applyConst y applyDest para aplicar transformaciones sobre listas. 

public class Transformers {
    public static <T> ArrayList<T> applyConst(Transformer<T> tran, ArrayList<T> a) {
        ArrayList<T> b = new ArrayList<>();
        for (T t : a) b.add(tran.transform(t));
        return b;
    }
}
